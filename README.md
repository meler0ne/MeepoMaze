# Meepo Maze

**Meepo Maze** — это игра-лабиринт, где геомант Meepo ищет путь к своим братьям. Лабиринты генерируются автоматически, можно редактировать стены мышью, сохранять и загружать лабиринты.

## Управление

| Клавиша | Действие |
|---------|----------|
| `1` | Классический лабиринт |
| `2` | Прямой путь |
| `+` / `-` | Увеличить/уменьшить размер |
| `Пробел` | Найти путь (BFS) |
| `C` | Очистить путь |
| `S` | Сохранить |
| `L` | Загрузить |
| `ESC` | Выход |

**Мышь:** 
- Левая кнопка — переключить стену
- Правая кнопка — удалить стену
